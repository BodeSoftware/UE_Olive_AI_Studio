## C++ Patterns

### UCLASS Macros
- UCLASS(Blueprintable) for classes that can be subclassed in Blueprint
- UCLASS(Abstract) for base classes that shouldn't be instantiated
- UCLASS(NotBlueprintable) for internal-only classes

### UPROPERTY Specifiers
- EditAnywhere, BlueprintReadWrite — designer-tweakable in BP
- VisibleAnywhere, BlueprintReadOnly — visible but not editable
- Category="GroupName" — organize in details panel
- meta=(ClampMin, ClampMax) — constrain numeric values
- Transient — not saved to disk

### UFUNCTION Specifiers
- BlueprintCallable — callable from Blueprint graphs
- BlueprintPure — no side effects, can be called freely
- BlueprintImplementableEvent — overridable in Blueprint
- BlueprintNativeEvent — has C++ default, overridable in BP
- Server/Client/NetMulticast — network replication

### Naming Conventions
- Classes: UMyComponent, AMyActor, FMyStruct
- Booleans: bIsActive, bHasTarget, bCanFire
- Delegates: FOnDamageReceived, FOnHealthChanged
- Interfaces: IMyInterface (pure C++), UMyInterface (UE reflected)
