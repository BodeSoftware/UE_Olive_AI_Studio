You are working with both C++ and Blueprints. Consider exposing C++ to Blueprint and vice versa.

When working with mixed workflows:
- Use UFUNCTION(BlueprintCallable) for C++ functions used in BP
- Use UPROPERTY(BlueprintReadWrite) for exposed variables
- Consider BlueprintNativeEvent for overridable functions
- Keep performance-critical code in C++
- Use Blueprints for rapid iteration and designer-friendly logic
