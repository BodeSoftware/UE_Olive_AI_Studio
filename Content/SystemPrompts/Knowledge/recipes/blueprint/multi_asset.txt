MULTI-ASSET workflow (e.g. gun + bullet, spawner + enemy):

General multi-asset sequence:
1. Create ALL asset structures first (use olive.build to batch: create + scaffold)
2. Wire graph logic per asset (use olive.build with apply_plan_json steps)
3. Compile all assets at the end

Create structures before graph logic so cross-refs work
(BP_Gun can reference BP_Bullet_C as a type).
Blueprint class refs append _C: "BP_Bullet_C" not "BP_Bullet".
Complete one asset's logic fully before the next.
For each olive.build call, use on_error:"continue_remaining" so partial successes aren't lost.
