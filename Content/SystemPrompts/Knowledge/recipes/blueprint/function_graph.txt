TAGS: function graph entry exec_after first step graph_target
---
Function graphs have an auto-created entry node. Do NOT add an event step.
First impure step needs no exec_after (auto-chains from entry).
Subsequent impure steps MUST have exec_after.
graph_target in apply_plan_json = function name, e.g. "Fire".
Return values: add a return step with op:"return" and the return pin name.
Create the function first with blueprint.add_function before applying plan_json.
Read with blueprint.read_function -> {path, function_name:"Fire"} to inspect existing logic.
