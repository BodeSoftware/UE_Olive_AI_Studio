Edit an existing graph:
1. Read the graph first (blueprint.read section:"graph" or blueprint.read_function)
2. Choose approach by scope:
   - Small change (1-3 nodes): remove_node + add_node + connect_pins on specific nodes
   - Add logic alongside existing: apply_plan_json mode:"merge" preserves existing nodes
   - Rewrite function: apply_plan_json mode:"replace" graph_target:"FuncName" — clears non-entry nodes atomically

NEVER loop remove_node to clear a graph. Use mode:"replace" instead.

Node IDs come from read results — never guess them.
For function graphs, use graph_target set to the function name.
