Add variables and components:
  blueprint.add_variable -> {path, variable:{name, type, default_value, flags}}
  blueprint.add_component -> {path, class, name, parent}
  blueprint.modify_component -> {path, name, properties:{...}}
Batch all in one turn. Create components first if variables reference them.

Variable type shorthand works for basics: "Float", "Boolean", "Integer", "String", "Vector", "Rotator", "Transform".
For complex types see the object_variable_type recipe.
Component property values use UE format: Vectors "(X=1,Y=2,Z=3)" not JSON.
Object refs use class_name, not sub_category or asset paths.
Blueprint class refs MUST append _C: "BP_Gun_C" not "BP_Gun".
