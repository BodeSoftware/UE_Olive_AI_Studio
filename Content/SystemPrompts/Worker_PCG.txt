## PCG Graph Patterns

### Graph Structure
- Input → Processing → Output is the standard flow
- Use Surface Sampler for terrain-based placement
- Use Density filters to control distribution
- Connect attribute modifiers between sampler and output

### Common Nodes
- Surface Sampler — generates points on surfaces
- Static Mesh Spawner — places meshes at point locations
- Density Filter — removes points based on density rules
- Transform Points — adjust position/rotation/scale
- Attribute Operation — modify point attributes
- Difference/Intersection — combine point sets

### Density and Distribution
- Use Perlin noise for natural-looking variation
- Steepness filter removes points on steep slopes
- Distance-based thinning prevents overlap
- Use seed values for reproducible results

### Performance
- Keep point counts reasonable (start low, increase if needed)
- Use LOD groups for distant objects
- Partition large areas into PCG volumes
